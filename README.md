**Архитектурный проект: Plate of Fate (Тарелка судьбы)**

**Описание проекта**
Текстовый квест про повара, который хочет уволиться, но шеф не отпускает его, пока тот не отработает последнюю смену

**Состав архитектурного проекта**
В проект включены 4 UML-диаграммы:

1. Диаграмма классов — основная структура игры: классы, перечисления, связи (композиция, агрегация, наследование, зависимости).
2. Диаграмма состояний — конечный автомат игры: 3 этажа, офис шефа, 4 концовки, условия переходов.
3. Диаграмма последовательности — полный цикл обслуживания клиента: от выбора до подачи блюда, с ветвлением для правильного/неправильного рецепта.
4. Диаграмма компонентов — 4-слойная архитектура (Presentation, Application, Domain, Infrastructure) с интерфейсами.
